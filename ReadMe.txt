HOW TO RUN:

1. First  make sure javafx is installed , if not
	Install openjfx, open terminal and type: sudo apt install openjfx

2. Please install the font provided within the tatai folder, "Maiandra GD Regular.ttf",
	To Install, double click the ttf file and click install in the top right corner.

3. Last, run Tatai! by opening the terminal and using the cd command, 
 	going to the same folder as the jar and type: java -jar tataiFinal.jar.
